01 — Filosofi

Hak cipta adalah masalah arsitektur.

Sebagian besar terjemahan Alkitab — termasuk Terjemahan Baru — berstatus berhak cipta. Proyek ini menanggapinya bukan dengan mengabaikan, melainkan dengan merancang ulang: teks bawaan adalah domain-publik, akses ke terjemahan berhak cipta bersifat opt-in dan hanya terjadi saat runtime.

i.

Bring Your Own Data

Mesin tidak membawa teks berhak cipta. Cukup letakkan berkas JSON di ALKITAB_DATA_DIR dan ia langsung terbaca di samping contoh domain-publik yang sudah tertanam.

ii.

Scrape adalah opt-in

Akses ke Terjemahan Baru tetap mungkin lewat adapter scrape yang diaktifkan dengan ALKITAB_SCRAPE=1 — sekadar proxy saat runtime, tanpa redistribusi.

iii.

Tiga lapis, satu arah

Transport, mesin, adapter. Dependensi hanya mengalir ke dalam. Itu yang membuat BYOD dan opt-in scrape bekerja tanpa menulis ulang inti.

03 — Siklus permintaan

Bagaimana satu GET menjadi ayat.

Dari path hingga JSON, setiap langkah melalui satu titik: Engine.

i

Menerima GET /v1/kjv/3john/1

Router Go 1.22 menangkap path values. {chapter} divalidasi numerik; bukan → 400.

ii

Resolve book → id

ResolveBookID menerima id, nama Inggris, atau nama Indonesia (case-insensitive) dan menormalisasinya ke "3john".

iii

Engine mengecek cache

Cache in-memory version:book:chapter dijamin RWMutex. Dudukan → langsung kembali. Luput → lanjut ke source.

iv

Source menjawab (local, atau Chain → scrape)

local.Chapter membaca JSON tertanam. Jika versi tak ada dan scrape aktif, Chain meneruskan ke adapter scrape.

v

JSON keluar

Hasil di-cache, diserialisasi. Error bertipe memetakan ke kode status yang tepat — tak ada pesan internal yang bocor.

04 — Endpoint

Tujuh rute, kontrak sederhana.

JSON masuk, JSON keluar. {book} menerima id, nama, atau singkatan — Inggris maupun Indonesia.

GET
/v1/translations
Daftar terjemahan

Terjemahan yang tersedia di source aktif (contoh tertanam + direktori data).

GET
/v1/{version}/books
Struktur kitab

Daftar kitab, jumlah pasal, dan kategori PL/PB untuk satu terjemahan.

GET
/v1/{version}/{book}/{chapter}
Satu pasal

Seluruh ayat, termasuk judul bagian.

GET
/v1/{version}/{book}/{chapter}/{verse}
Satu ayat

Disaring ke satu nomor ayat.

GET
/v1/search?q={q}&version={v}
Pencarian teks

Substring case-insensitive di korpus lokal.

GET
/v1/daily?version={v}
Ayat hari ini

Deterministik per tanggal & versi (UTC).

GET
/v1/random?version={v}
Ayat acak

Seed otomatis dari math/rand/v2.

06 — Bring Your Own Data

“Tidak ada teks berhak cipta di repositori ini — dan sengaja demikian.”

Contoh tertanam adalah domain-publik (KJV). Terjemahan Berhak Cipta seperti TB © LAI hanya bisa diakses saat runtime lewat proxy scrape — tanggung jawab operator, bukan proyek.

alkitab-data/tb.json
// format BYOD — satu file per terjemahan
{
  "translation": {
    "id": "tb",
    "name": "Terjemahan Baru",
    "language": "id"
  },
  "books": [
    { "id": "john",
      "name": "Yohanes",
      "chapters": 21,
      "chapter_data": [
        { "number": 1,
          "verses": [
            { "verse": 1,
              "type": "content",
              "content": "…" }
          ] }
      ] }
  ]
}
